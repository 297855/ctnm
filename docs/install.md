# 安装

## 安装包安装

[releases](https://github.com/djylb/nps/releases)

下载对应的系统版本即可，服务端和客户端是单独的

## 源码安装

- 安装源码
  ```go get -u github.com/djylb/nps```
- 编译

服务端```go build cmd/nps/nps.go```

客户端```go build cmd/npc/npc.go```
